## 1. Помощь внутри Neovim

| Действие | Команда |
|---|---|
| Открыть справку по теме | `:help <тема>` (например `:help split`) |
| Открыть оглавление справки | `:help` |
| Закрыть окно справки | `:q` |
| Fuzzy-поиск по всей справке через Telescope | `<leader>fh` |
| Список всех настроенных хоткеев с поиском | `<leader>fk` |
| Подсказка после нажатия `<leader>` | автоматически — popup which-key через секунду |

**Правило:** забыл хоткей — нажми `<leader>` и подожди. Появится popup со всеми вариантами. Нажми букву группы (например `f`) — popup перерисуется и покажет все варианты `<leader>f...`.

## 2. Режимы (modes)

Neovim — модальный редактор. В разных режимах одни и те же клавиши делают разное.

| Режим | Назначение | Как войти |
|---|---|---|
| **Normal** | навигация, команды | `<Esc>` (всегда возвращает сюда) |
| **Insert** | обычный набор текста | `i`, `a`, `o` (из normal) |
| **Visual** | выделение текста | `v`, `V`, `<C-v>` (из normal) |
| **Command** | ввод Ex-команды (`:w`, `:Lazy`) | `:` (из normal) |
| **Terminal** | работа во встроенном терминале | автоматически |

**Главное правило новичка:** если что-то идёт не так — нажми `<Esc>`. Это всегда возвращает в Normal mode.

## 3. Базовая навигация (в Normal mode)

### По символам и словам

| Клавиша | Действие |
|---|---|
| `h` | влево (один символ) |
| `j` | вниз (одна строка) |
| `k` | вверх (одна строка) |
| `l` | вправо (один символ) |
| `w` | в начало следующего слова |
| `b` | в начало предыдущего слова |
| `e` | в конец текущего/следующего слова |

### По строке

| Клавиша | Действие |
|---|---|
| `0` | в начало строки (нулевая колонка) |
| `^` | к первому непустому символу строки |
| `$` | в конец строки |
| `f<char>` | прыгнуть к следующему символу `<char>` в строке |
| `t<char>` | прыгнуть **перед** следующим символом `<char>` |

### По файлу

| Клавиша | Действие |
|---|---|
| `gg` | в начало файла |
| `G` | в конец файла |
| `<число>G` | на строку с номером (например `42G`) |
| `<C-u>` | вверх на полэкрана |
| `<C-d>` | вниз на полэкрана |
| `<C-b>` | вверх на полный экран |
| `<C-f>` | вниз на полный экран |
| `%` | прыгнуть к парной скобке |

### Поиск по файлу

| Клавиша | Действие |
|---|---|
| `/<текст>` | поиск вперёд |
| `?<текст>` | поиск назад |
| `n` | следующее совпадение |
| `N` | предыдущее совпадение |
| `*` | искать слово под курсором (вперёд) |
| `#` | искать слово под курсором (назад) |

## 4. Базовое редактирование

### Вход в Insert mode

| Клавиша | Действие |
|---|---|
| `i` | вставка перед курсором |
| `a` | вставка после курсора |
| `I` | вставка в начале строки |
| `A` | вставка в конце строки |
| `o` | новая строка ниже + insert |
| `O` | новая строка выше + insert |

### Удаление и замена

| Клавиша | Действие |
|---|---|
| `x` | удалить символ под курсором |
| `dd` | удалить строку (можно потом вставить через `p`) |
| `dw` | удалить слово |
| `d$` или `D` | удалить до конца строки |
| `cw` | удалить слово и войти в insert |
| `cc` | удалить строку и войти в insert |
| `r<char>` | заменить один символ под курсором |
| `R` | войти в режим замены (overwrite) |

### Копирование и вставка

| Клавиша | Действие |
|---|---|
| `yy` | копировать строку (yank) |
| `yw` | копировать слово |
| `y$` | копировать до конца строки |
| `p` | вставить **после** курсора |
| `P` | вставить **перед** курсором |

### Отмена и повтор

| Клавиша | Действие |
|---|---|
| `u` | отменить (undo) |
| `<C-r>` | повторить отменённое (redo) |
| `.` | повторить последнее действие (любое) |

## 5. Visual Mode (выделение)

| Клавиша | Действие |
|---|---|
| `v` | выделение по символам |
| `V` | выделение целыми строками |
| `<C-v>` | блочное выделение (прямоугольник) |
| `y` | копировать выделенное |
| `d` | вырезать выделенное |
| `c` | удалить выделенное и войти в insert |
| `>` | сдвинуть выделение вправо (увеличить отступ) |
| `<` | сдвинуть выделение влево (уменьшить отступ) |
| `=` | автоформатирование выделения |
| `<Esc>` | выйти из visual mode |

## 6. Окна и сплиты

**Окно (window)** = область экрана с файлом. Экран можно делить на несколько окон.

#### Создать
| Команда                | Действие                                            |
|------------------------|-----------------------------------------------------|
| `:split` / `:sp`       | горизонтальный сплит, тот же файл сверху            |
| `:vsplit` / `:vsp`     | вертикальный сплит, тот же файл справа              |
| `:split <путь>`        | открыть другой файл в горизонтальном сплите         |
| `:vsplit <путь>`       | открыть другой файл в вертикальном сплите           |
| `<C-w> s`              | то же, что `:split` (быстрая клавиатурная версия)   |
| `<C-w> v`              | то же, что `:vsplit`                                |

#### Переключиться
| Биндинг   | Действие                |
|-----------|-------------------------|
| `<C-w> h` | в окно слева            |
| `<C-w> j` | в окно ниже             |
| `<C-w> k` | в окно выше             |
| `<C-w> l` | в окно справа           |
| `<C-w> w` | в следующее окно        |

#### Управлять
| Биндинг   | Действие                              |
|-----------|---------------------------------------|
| `<C-w> q` | закрыть текущее окно                  |
| `<C-w> c` | то же самое, что `<C-w> q`            |
| `<C-w> o` | оставить только текущее, остальные закрыть |
| `<C-w> =` | выровнять все окна по размеру         |
| `<C-w> _` | максимизировать по высоте             |
| `<C-w> |` | максимизировать по ширине             |

## 7. Буферы и сохранение

**Буфер** = открытый файл в памяти.

| Команда | Действие |
|---|---|
| `:e <файл>` | открыть файл (или создать новый) |
| `:w` | сохранить |
| `:q` | закрыть текущее окно |
| `:wq` | сохранить и закрыть |
| `:q!` | закрыть без сохранения (потерять изменения) |
| `:wa` | сохранить все открытые файлы |
| `:qa` | закрыть всё |
| `:bd` | закрыть текущий буфер |
| `:bn` или `<S-l>` | следующий буфер |
| `:bp` или `<S-h>` | предыдущий буфер |
| `:ls` | список всех открытых буферов |
